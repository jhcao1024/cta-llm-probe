# Artifacts

This directory holds intermediate files shared between experiments, as some experiments rely on the output of previous ones.

The subdirectories are empty by default and populated incrementally as you run the experiments.

Some of the LLM/API-generated outputs are released under `../data/released`.

## Layout

```text
artifacts/
├── failure-patterns/
├── representation-bias/
├── retrieval-diagnostics/
└── binary-probing/
```
